# ZX Control — Setup (modelo)

> Este é o **repositório-modelo** dos Setups do ZX Control. Quando criado a partir do
> botão **"Use this template"**, vira um Setup novo. O Claude personaliza o conteúdo
> seguindo o `CLAUDE.md`.

## O que você vai conseguir no final

_(O Claude preenche esta seção com o resultado concreto do setup — em benefício, não em termos técnicos.)_

## Como começar

```bash
git clone https://github.com/zxmarketingdigital/zx-control-setup{N}-{slug}.git
cd zx-control-setup{N}-{slug}
claude
```

Depois é só dizer ao Claude: **"vamos começar o setup"**. Ele conduz o resto.

## Pré-requisitos

- Claude Code instalado
- Conta GitHub (para clonar)

## Estrutura

| Arquivo | Para quê |
|---------|----------|
| `setup/check_prerequisites.py` | Valida o ambiente do aluno (1ª coisa que roda) |
| `setup/setup_*.py` | Uma etapa de instalação cada |
| `docs/index.html` | Dashboard local do setup |
| `MASTERCLASS.md` | Roteiro da aula |
